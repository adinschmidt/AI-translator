# البدء

ابدأ الاستخدام في بضع دقائق.

## 1) تثبيت الامتداد

- [Chrome Web Store](https://chromewebstore.google.com/detail/jabhdcjhdlnppcpbdghnkfkdpfcfleba)
- [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/ai-translator/)
- [تعليمات التثبيت اليدوي](/ar/installation)

## 2) قم بتكوين المزود الخاص بك

1. افتح صفحة الامتداد **الخيارات**.
2. اختر **أساسي** (الإعدادات الافتراضية البسيطة) أو **متقدم** (التحكم الكامل).
3. أضف مفتاح API الخاص بك واحفظه.

هل تحتاج إلى مفتاح؟ راجع [الموفرون ومفاتيح واجهة برمجة التطبيقات](/ar/providers).

## 3) ترجمة شيء ما

- حدد النص ← انقر بزر الماوس الأيمن ← **ترجمة النص المحدد**
- انقر بزر الماوس الأيمن في أي مكان → **ترجمة الصفحة**

إذا كان هناك أي شيء يبدو خاطئًا، فراجع صفحة [استكشاف الأخطاء وإصلاحها](/ar/troubleshooting).
