# الاستخدام

## ترجمة النص المحدد

1. قم بتمييز النص في أي صفحة ويب.
2. انقر بزر الماوس الأيمن → **ترجمة النص المحدد**.
3. تظهر الترجمة في نافذة منبثقة.

## ترجمة صفحة كاملة

1. انقر بزر الماوس الأيمن في أي مكان بالصفحة.
2. اختر **ترجمة الصفحة**.
3. يستبدل الامتداد نص الصفحة بالمخرجات المترجمة.

## إظهار زر الترجمة المضمنة

يمكنك اختياريًا إظهار زر **ترجمة** صغير عند تحديد النص.
قم بتمكين هذا في **الإعدادات → إظهار زر الترجمة عند التحديد**.
