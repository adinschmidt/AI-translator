# سياسة الخصوصية لـ AI Translator

**آخر تحديث:** 10 يناير 2026

## 1. نظرة عامة

AI Translator ("نحن" أو "خاصتنا" أو "الامتداد") هو امتداد للمتصفح من جانب العميل
يتيح للمستخدمين ترجمة النص باستخدام مفاتيح AI API الخاصة بهم. التمديد
يعمل بالكامل داخل متصفحك ولا ينقل البيانات إلى أي خوادم
يملكها أو يديرها المطور.

**هذه ليست نصيحة قانونية.** توضح هذه السياسة كيفية تعامل الإضافة
البيانات لمساعدتك على اتخاذ قرار مستنير بشأن استخدامها.

## 2. جمع البيانات وتخزينها

- **البيانات الشخصية:** نحن (المطور) لا نقوم بجمع أو تخزين أو الوصول إليها
  أي بيانات شخصية أو نص مترجم أو سجل التصفح.
- **مفاتيح API:** يتم تخزين مفاتيح API الخاصة بك حصريًا على جهازك باستخدام
  التخزين المتزامن للمتصفح (`browser.storage.sync`). هذا يعني الخاص بك
  تتم مزامنة الإعدادات عبر المتصفحات الخاصة بك عند تسجيل الدخول، ولكن لا يتم ذلك أبدًا
  تنتقل إلى المطور أو أي طرف ثالث بخلاف الذكاء الاصطناعي المحدد
  المزود الذي تختاره لطلبات الترجمة.
- **لا يوجد خادم خلفي:** لا يعمل هذا الامتداد على خادم خلفي. الكل
  تتم المعالجة محليًا في متصفحك.

## 3. مشاركة البيانات مع أطراف ثالثة

لكي يعمل، يرسل هذا الامتداد النص الذي تحدده بوضوح للترجمة
مباشرة من متصفحك إلى موفر الذكاء الاصطناعي الذي قمت بتكوينه. باستخدام
بهذا التمديد، فإنك تقر بأن بياناتك تخضع للخصوصية
سياسات هؤلاء مقدمي الخدمة:

| مقدم                 | سياسة الخصوصية                                                                     |
| -------------------- | ---------------------------------------------------------------------------------- |
| **OpenAI**           | [https://openai.com/privacy](https://openai.com/privacy)                           |
| **Anthropic**        | [https://www.anthropic.com/legal/privacy](https://www.anthropic.com/legal/privacy) |
| **Google (Gemini)** | [https://ai.google.dev/gemini-api/terms](https://ai.google.dev/gemini-api/terms)   |
| **xAI (Grok)**       | [https://x.ai/legal/privacy-policy](https://x.ai/legal/privacy-policy)             |
| **OpenRouter**       | [https://openrouter.ai/privacy](https://openrouter.ai/privacy)                     |
| **Ollama (محلي)**    | تبقى البيانات على جهازك المحلي؛ لا يوجد نقل خارجي.                                 |

** اعتبارات هامة: **

- قد يستخدم بعض مقدمي الخدمة بياناتك لتدريب نماذجهم ما لم تقم بإلغاء الاشتراك
  (راجع سياسة كل مزود).
- OpenRouter يقوم بتوجيه الطلبات إلى العديد من موفري الذكاء الاصطناعي الأساسيين، كل منهم مع
  سياسات البيانات الخاصة.
- نحن لا نبيع أو نتاجر أو ننقل بياناتك إلى أطراف خارجية.

## 4. سياسات متجر المتصفح

عند توزيعه من خلال متاجر ملحقات المتصفح، يلتزم هذا الملحق بـ
سياسات بيانات المستخدم المعمول بها، بما في ذلك [Chrome Web Store سياسة بيانات المستخدم](https://developer.chrome.com/docs/webstore/program-policies/) و[سياسات الوظائف الإضافية لـ Firefox](https://extensionworkshop.com/documentation/publish/add-on-policies/).

## 5. شرح الأذونات

يطلب هذا الملحق الأذونات التالية:

| إذن                               | الغرض                                                                                             |
| --------------------------------- | ------------------------------------------------------------------------------------------------- |
| `contextMenus`                    | إضافة خيار "ترجمة" إلى قائمة النقر بزر الماوس الأيمن                                              |
| `scripting`                       | يُدخل تراكب واجهة مستخدم الترجمة على صفحات الويب                                                  |
| `storage`                         | يحفظ مفاتيح API وتفضيلاتك محليًا                                                                  |
| `host_permissions` (`<all_urls>`) | مطلوب لعرض الترجمات على أي صفحة ويب تزورها ولإجراء مكالمات API لمزود الذكاء الاصطناعي الذي اخترته |

## 6. التحكم في المستخدم

- **الحذف:** يمكنك إزالة جميع البيانات المخزنة بواسطة هذا الامتداد عن طريق إلغاء التثبيت
الامتداد أو مسح بيانات الامتداد في متصفحك.

- **إلغاء الاشتراك:** لا يتم إرسال أي بيانات إلى موفري خدمات الذكاء الاصطناعي ما لم تقم بشكل صريح بتشغيل ملف
  عمل الترجمة.
- **اختيار الموفر:** يمكنك التحكم في موفر الذكاء الاصطناعي الذي يتلقى بياناتك من خلاله
  تحديده في إعدادات التمديد.

## 7. الأمن

- يتم تخزين مفاتيح API في وحدة التخزين الآمنة للمتصفح ويتم نقلها فقط
  عبر HTTPS إلى نقاط نهاية موفر الذكاء الاصطناعي.
- لا يقوم الامتداد بتسجيل أي نص مترجم أو تخزينه مؤقتًا أو الاحتفاظ به بعد ذلك
  عرضه.
