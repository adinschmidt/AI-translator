## ما هو AI Translator؟

AI Translator هو امتداد لمتصفح Chrome وFirefox يترجم النص المحدد أو
صفحات كاملة باستخدام موفري الذكاء الاصطناعي القابلين للتكوين. يمكنك التحكم في النموذج، ونقطة النهاية،
مفتاح API وتعليمات الترجمة.
