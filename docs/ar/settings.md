# الإعدادات

افتح قائمة الامتدادات وانقر فوق **خيارات** لتكوين AI Translator.

## الأوضاع

### الوضع الأساسي

- اختر موفرًا (OpenAI أو Anthropic أو Google).
- أدخل مفتاح API.
- اختر لغتك المستهدفة.
- يستخدم الامتداد نقاط النهاية والنماذج الموصى بها تلقائيًا.

### الوضع المتقدم

- اختر من بين جميع مقدمي الخدمات المدعومين.
- تخصيص نقطة نهاية API واسم النموذج وتعليمات الترجمة.
- مثالي للإعدادات ذاتية الاستضافة أو المتوافقة مع OpenAI أو إعدادات المستخدم القوي.

## تعليمات الترجمة

استخدم **تعليمات الترجمة** للتحكم في النغمة والأسلوب. النص المحدد هو
يتم إلحاقه تلقائيًا. مثال:

```
Translate to Spanish. Keep the tone friendly and concise.
```

## نقطة نهاية ونموذج واجهة برمجة التطبيقات

لدى كل مزود خيار **Fill Default** الذي يستعيد الموصى به
نقطة النهاية واسم النموذج. يمكنك تجاوز كليهما في الوضع المتقدم إذا لزم الأمر.

## التخزين والخصوصية

يتم تخزين الإعدادات في `chrome.storage.sync` بحيث تستمر عبر الأجهزة.
لا يتم تسجيل مفاتيح واجهة برمجة التطبيقات (API) مطلقًا في وحدة التحكم.
