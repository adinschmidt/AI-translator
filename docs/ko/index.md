## AI Translator란 무엇인가요?

AI Translator은 선택한 텍스트를 번역하는 Chrome 및 Firefox 확장 프로그램입니다.
구성 가능한 AI 공급자를 사용하여 전체 페이지. 모델, 엔드포인트,
API 키 및 번역 지침.
