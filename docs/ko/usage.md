# 사용법

## 선택한 텍스트 번역

1. 웹페이지의 텍스트를 강조표시합니다.
2. 마우스 오른쪽 버튼 클릭 → **선택한 텍스트 번역**.
3. 번역이 팝업으로 나타납니다.

## 전체 페이지 번역

1. 페이지의 아무 곳이나 마우스 오른쪽 버튼으로 클릭하세요.
2. **페이지 번역**을 선택합니다.
3. 확장 기능은 페이지 텍스트를 번역된 출력으로 바꿉니다.

## 인라인 번역 버튼 표시

텍스트를 선택할 때 선택적으로 작은 **번역** 버튼을 표시할 수 있습니다.
**설정 → 선택 시 번역 버튼 표시**에서 이 기능을 활성화하세요.
