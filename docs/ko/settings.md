# 설정

확장 메뉴를 열고 **옵션**을 클릭하여 AI Translator를 구성합니다.

## 모드

### 기본 모드

- 제공자를 선택하십시오(OpenAI, Anthropic 또는 Google).
- API 키를 입력하세요.
- 타겟 언어를 선택하세요.
- 확장 프로그램은 권장 엔드포인트와 모델을 자동으로 사용합니다.

### 고급 모드

- 지원되는 모든 제공업체 중에서 선택하세요.
- API 엔드포인트, 모델 이름 및 번역 지침을 사용자 정의합니다.
- 자체 호스팅, OpenAI 호환 또는 고급 사용자 설정에 이상적입니다.

## 번역 지침

**번역 지침**을 사용하여 어조와 스타일을 제어하세요. 선택한 텍스트는
자동으로 추가됩니다. 예:

```
Translate to Spanish. Keep the tone friendly and concise.
```

## API 엔드포인트 및 모델

각 공급자에는 권장 사항을 복원하는 **기본값 채우기** 옵션이 있습니다.
엔드포인트 및 모델 이름. 필요한 경우 고급 모드에서 두 가지를 모두 재정의할 수 있습니다.

## 저장 및 개인정보 보호

설정은 `chrome.storage.sync`에 저장되므로 여러 장치에 걸쳐 지속됩니다.
API 키는 콘솔에 기록되지 않습니다.
