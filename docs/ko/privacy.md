# AI Translator 개인정보 보호정책

**최종 업데이트:** 2026년 1월 10일

## 1. 개요

AI Translator("당사", "당사의" 또는 "확장 프로그램")은 클라이언트측 브라우저 확장입니다.
사용자가 자신의 AI API 키를 사용하여 텍스트를 번역할 수 있습니다. 확장
전적으로 귀하의 브라우저 내에서 작동하며 어떤 서버에도 데이터를 전송하지 않습니다.
개발자가 소유하거나 운영합니다.

**이것은 법적 조언이 아닙니다.** 이 정책은 확장 프로그램이 처리하는 방법을 설명합니다.
정보에 입각한 사용 결정을 내리는 데 도움이 되는 데이터입니다.

## 2. 데이터 수집 및 저장

- **개인 데이터:** 당사(개발자)는 다음 항목을 수집, 저장 또는 액세스하지 않습니다.
  모든 개인 데이터, 번역된 텍스트 또는 검색 기록.
- **API 키:** 귀하의 API 키는 다음을 사용하여 귀하의 장치에만 저장됩니다.
  브라우저의 동기화 저장소(`browser.storage.sync`). 이는 귀하의
  설정은 로그인하면 브라우저 간에 동기화되지만 동기화되지는 않습니다.
  해당 AI가 아닌 개발자나 제3자에게 전송
  번역 요청을 위해 귀하가 선택한 제공업체.
- **백엔드 서버 없음:** 이 확장 프로그램은 백엔드 서버를 운영하지 않습니다. 모두
  처리는 브라우저에서 로컬로 수행됩니다.

## 3. 제3자와 데이터 공유

작동하려면 이 확장 프로그램이 번역을 위해 명시적으로 선택한 텍스트를 보냅니다.
귀하의 브라우저에서 귀하가 구성한 AI 제공업체로 직접 전송됩니다. 사용하여
이 확장 프로그램을 사용하면 귀하의 데이터가 개인정보 보호정책의 적용을 받는다는 점을 인정합니다.
해당 제공업체의 정책:

| 공급자                  | 개인정보 보호정책                                                                  |
| ----------------------- | ---------------------------------------------------------------------------------- |
| **OpenAI**              | [https://openai.com/privacy](https://openai.com/privacy)                           |
| **Anthropic**           | [https://www.anthropic.com/legal/privacy](https://www.anthropic.com/legal/privacy) |
| **Google (Gemini)** | [https://ai.google.dev/gemini-api/terms](https://ai.google.dev/gemini-api/terms)   |
| **xAI (Grok)**          | [https://x.ai/legal/privacy-policy](https://x.ai/legal/privacy-policy)             |
| **OpenRouter**          | [https://openrouter.ai/privacy](https://openrouter.ai/privacy)                     |
| **Ollama (로컬)**       | 데이터는 로컬 컴퓨터에 유지됩니다. 외부 전송 없음.                                 |

**중요 고려사항:**

- 일부 제공업체에서는 귀하가 탈퇴하지 않는 한 귀하의 데이터를 사용하여 모델을 교육할 수 있습니다.
  (각 제공업체의 정책을 확인하세요).
- OpenRouter은 요청을 다양한 기본 AI 제공자에게 라우팅합니다.
  자신의 데이터 정책.
- 당사는 귀하의 데이터를 외부 당사자에게 판매, 거래 또는 기타 방식으로 전송하지 않습니다.

## 4. 브라우저 스토어 정책

브라우저 확장 저장소를 통해 배포될 때 이 확장은 다음을 준수합니다.
[Chrome Web Store 사용자 데이터 정책](https://developer.chrome.com/docs/webstore/program-policies/) 및 [Firefox 추가 기능 정책](https://extensionworkshop.com/documentation/publish/add-on-policies/)을 포함한 적용 가능한 사용자 데이터 정책.

## 5. 권한 설명

이 확장 프로그램은 다음 권한을 요청합니다.

| 허가                              | 목적                                                                                                     |
| --------------------------------- | -------------------------------------------------------------------------------------------------------- |
| `contextMenus`                    | 마우스 오른쪽 버튼 클릭 메뉴에 "번역" 옵션 추가                                                          |
| `scripting`                       | 웹페이지에 번역 UI 오버레이 삽입                                                                         |
| `storage`                         | API 키와 기본 설정을 로컬에 저장                                                                         |
| `host_permissions` (`<all_urls>`) | 귀하가 방문하는 모든 웹페이지에 번역을 표시하고 귀하가 선택한 AI 공급자에게 API를 호출하는 데 필요합니다 |

## 6. 사용자 제어

- **삭제:** 이 확장 프로그램을 제거하면 이 확장 프로그램에 저장된 모든 데이터를 제거할 수 있습니다.
  확장 프로그램을 삭제하거나 브라우저의 확장 프로그램 데이터를 삭제하세요.

- **거부:** 명시적으로 트리거하지 않는 한 AI 제공업체에 데이터가 전송되지 않습니다.
  번역 작업.
- **제공업체 선택:** 귀하는 어느 AI 제공업체가 귀하의 데이터를 수신할지 제어합니다.
  확장 설정에서 선택하세요.

## 7. 보안

- API 키는 브라우저의 보안 저장소에 저장되며 전송만 가능합니다.
  HTTPS를 통해 AI 공급자 엔드포인트로 전송됩니다.
- 확장 프로그램은 이후 번역된 텍스트를 기록, 캐시 또는 유지하지 않습니다.
  그것을 표시합니다.
