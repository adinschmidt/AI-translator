# 시작하기

몇 분이면 시작할 수 있습니다.

## 1) 확장 프로그램 설치

- [Chrome Web Store](https://chromewebstore.google.com/detail/jabhdcjhdlnppcpbdghnkfkdpfcfleba)
- [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/ai-translator/)
- [수동설치 안내](/ko/installation)

## 2) 공급자 구성

1. 확장 **옵션** 페이지를 엽니다.
2. **기본**(간단한 기본값) 또는 **고급**(모든 권한)을 선택합니다.
3. API 키를 추가하고 저장합니다.

키가 필요하세요? [공급자 및 API 키](/ko/providers)를 참조하세요.

## 3) 무언가 번역하기

- 텍스트 선택 → 마우스 오른쪽 버튼 클릭 → **선택한 텍스트 번역**
- 아무 곳이나 마우스 오른쪽 버튼으로 클릭 → **페이지 번역**

문제가 있는 경우 [문제 해결](/ko/troubleshooting) 페이지를 확인하세요.
