# 设置

打开扩展菜单并单击“**选项**”以配置 AI Translator。

## 模式

### 基本模式

- 选择提供商（OpenAI、Anthropic 或 Google）。
- 输入 API 密钥。
- 选择您的目标语言。
- 扩展自动使用推荐的端点和模型。

### 高级模式

- 从所有受支持的提供商中进行选择。
- 自定义API端点、模型名称和翻译指令。
- 非常适合自托管、OpenAI 兼容或高级用户设置。

## 翻译说明

使用**翻译说明**来控制语气和风格。所选文本是
自动附加。示例：

```
Translate to Spanish. Keep the tone friendly and concise.
```

## API 端点和模型

每个提供商都有一个“填充默认值”选项，可恢复推荐的值
端点和模型名称。如果需要，您可以在高级模式下覆盖两者。

## 存储和隐私

设置存储在 `chrome.storage.sync` 中，因此它们可以跨设备保留。
API 密钥永远不会记录到控制台。
