## Что такое AI Translator?

AI Translator — это расширение Chrome и Firefox, которое переводит выделенный текст или
целые страницы с использованием настраиваемых поставщиков искусственного интеллекта. Вы управляете моделью, конечной точкой,
Ключ API и инструкции по переводу.
