# Настройки

Откройте меню расширения и нажмите **Параметры**, чтобы настроить AI Translator.

## Режимы

### Базовый режим

- Выберите поставщика (OpenAI, Anthropic или Google).
- Введите ключ API.
- Выберите целевой язык.
- Расширение автоматически использует рекомендуемые конечные точки и модели.

### Расширенный режим

- Выберите из всех поддерживаемых поставщиков.
- Настройте конечную точку API, название модели и инструкции по переводу.
- Идеально подходит для самостоятельного размещения, OpenAI-совместимого или опытного пользователя.

## Инструкции по переводу

Используйте **Инструкции по переводу**, чтобы контролировать тон и стиль. Выделенный текст
добавляется автоматически. Пример:

```
Translate to Spanish. Keep the tone friendly and concise.
```

## Конечная точка и модель API

У каждого поставщика есть опция **Заполнить по умолчанию**, которая восстанавливает рекомендованные значения.
конечная точка и имя модели. При необходимости вы можете переопределить оба параметра в расширенном режиме.

## Хранение и конфиденциальность

Настройки хранятся в `chrome.storage.sync`, поэтому они сохраняются на всех устройствах.
Ключи API никогда не регистрируются в консоли.
