# Политика конфиденциальности для AI Translator

**Последнее обновление:** 10 января 2026 г.

## 1. Обзор

AI Translator («мы», «наше» или «расширение») — это расширение браузера на стороне клиента.
это позволяет пользователям переводить текст, используя свои собственные ключи AI API. Расширение
работает полностью в вашем браузере и не передает данные ни на какие серверы
принадлежит или управляется разработчиком.

**Это не юридическая консультация.** Эта политика описывает, как расширение обрабатывает
данные, которые помогут вам принять обоснованное решение об их использовании.

## 2. Сбор и хранение данных

- **Личные данные:** Мы (разработчик) не собираем, не храним и не имеем доступа к
  любые личные данные, переведенный текст или историю просмотров.
- **Ключи API:** Ваши ключи API хранятся исключительно на вашем устройстве с помощью
  синхронизированное хранилище браузера (`browser.storage.sync`). Это означает, что ваш
  настройки синхронизируются в ваших браузерах при входе в систему, но они никогда не
  передается разработчику или любой третьей стороне, кроме конкретного ИИ
  поставщика, которого вы выбираете для запросов на перевод.
- **Без внутреннего сервера:** Это расширение не управляет внутренним сервером. Все
  обработка происходит локально в вашем браузере.

## 3. Обмен данными с третьими лицами

Для работы это расширение отправляет текст, который вы явно выбрали для перевода.
напрямую из вашего браузера к настроенному вами провайдеру искусственного интеллекта. Используя
это расширение, вы признаете, что на ваши данные распространяется политика конфиденциальности.
политика этих провайдеров:

| Провайдер             | Политика конфиденциальности                                                        |
| --------------------- | ---------------------------------------------------------------------------------- |
| **OpenAI**            | [https://openai.com/privacy](https://openai.com/privacy)                           |
| **Anthropic**         | [https://www.anthropic.com/legal/privacy](https://www.anthropic.com/legal/privacy) |
| **Google (Gemini)** | [https://ai.google.dev/gemini-api/terms](https://ai.google.dev/gemini-api/terms)   |
| **xAI (Grok)**        | [https://x.ai/legal/privacy-policy](https://x.ai/legal/privacy-policy)             |
| **OpenRouter**        | [https://openrouter.ai/privacy](https://openrouter.ai/privacy)                     |
| **Ollama (местный)**  | Данные остаются на вашем локальном компьютере; нет внешней передачи.               |

**Важные соображения:**

- Некоторые поставщики могут использовать ваши данные для обучения своих моделей, если вы не откажетесь.
  (проверьте политику каждого провайдера).
- OpenRouter направляет запросы различным базовым поставщикам ИИ, каждый со своими
  собственная политика данных.
- Мы не продаем, не обмениваем и не передаем иным образом ваши данные сторонним лицам.

## 4. Политики магазина браузера

При распространении через магазины расширений браузера это расширение соответствует
применимые политики пользовательских данных, включая [Chrome Web Store Политику пользовательских данных](https://developer.chrome.com/docs/webstore/program-policies/) и [Дополнительные политики Firefox](https://extensionworkshop.com/documentation/publish/add-on-policies/).

## 5. Объяснение разрешений

Это расширение запрашивает следующие разрешения:

| Разрешение                        | Цель                                                                                                                                 |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| `contextMenus`                    | Добавляет опцию «Перевести» в контекстное меню                                                                                       |
| `scripting`                       | Внедряет наложение пользовательского интерфейса перевода на веб-страницы                                                             |
| `storage`                         | Сохраняет ваши ключи API и настройки локально                                                                                        |
| `host_permissions` (`<all_urls>`) | Требуется для отображения переводов на любой посещаемой вами веб-странице и для выполнения вызовов API выбранного вами поставщика ИИ |

## 6. Пользовательский контроль

- **Удаление:** вы можете удалить все данные, хранящиеся в этом расширении, удалив его.

расширение или очистка данных расширения вашего браузера.

- **Отказ:** никакие данные не отправляются поставщикам ИИ, если вы явно не инициируете
  переводческое действие.
- **Выбор поставщика.** Вы сами определяете, какой поставщик ИИ будет получать ваши данные.
  выбрав его в настройках расширения.

## 7. Безопасность

- Ключи API хранятся в безопасном хранилище браузера и передаются только
  через HTTPS к конечным точкам поставщика ИИ.
- Расширение не регистрирует, не кэширует и не сохраняет переведенный текст после
  показывая это.

## 8. Изменения в настоящей политике

Мы можем время от времени обновлять данную политику конфиденциальности. Изменения будут отражены
в дате «Последнее обновление» вверху этого документа.
