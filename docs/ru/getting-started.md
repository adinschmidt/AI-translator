# Начало работы

Приступайте к работе за несколько минут.

## 1) Установите расширение

- [Chrome Web Store](https://chromewebstore.google.com/detail/jabhdcjhdlnppcpbdghnkfkdpfcfleba)
- [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/ai-translator/)
- [Инструкция по ручной установке](/ru/installation)

## 2) Настройте своего провайдера

1. Откройте страницу **Параметры** расширения.
2. Выберите **Базовый** (простые настройки по умолчанию) или **Расширенный** (полный доступ).
3. Добавьте свой ключ API и сохраните.

Нужен ключ? См. [Поставщики и ключи API](/ru/providers).

## 3) Перевести что-нибудь

- Выделите текст → щелкните правой кнопкой мыши → **Перевести выделенный текст**
  – Щелкните правой кнопкой мыши в любом месте → **Перевести страницу**

Если что-то не так, проверьте страницу [Устранение неполадок](/ru/troubleshooting).
