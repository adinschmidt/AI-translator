# Использование

## Перевести выделенный текст

1. Выделите текст на любой веб-странице.
2. Щелкните правой кнопкой мыши → **Перевести выделенный текст**.
3. Перевод появится во всплывающем окне.

## Перевести всю страницу

1. Щелкните правой кнопкой мыши в любом месте страницы.
2. Выберите **Перевести страницу**.
3. Расширение заменяет текст страницы переведенным выводом.

## Показать встроенную кнопку перевода

При желании вы можете отображать небольшую кнопку **Перевести** при выделении текста.
Включите это в **Настройки → Показывать кнопку «Перевести» при выборе**.
