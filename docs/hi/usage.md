# उपयोग

## चयनित पाठ का अनुवाद करें

1. किसी भी वेबपेज पर टेक्स्ट को हाइलाइट करें।
2. राइट-क्लिक करें → **चयनित टेक्स्ट का अनुवाद करें**।
3. अनुवाद एक पॉपअप में दिखाई देता है.

## पूरे पृष्ठ का अनुवाद करें

1. पेज पर कहीं भी राइट-क्लिक करें।
2. **अनुवाद पृष्ठ** चुनें।
3. एक्सटेंशन पेज टेक्स्ट को अनुवादित आउटपुट से बदल देता है।

## इनलाइन अनुवाद बटन दिखाएँ

टेक्स्ट चयनित होने पर आप वैकल्पिक रूप से एक छोटा **अनुवाद** बटन दिखा सकते हैं।
इसे **सेटिंग्स → चयन पर अनुवाद बटन दिखाएं** में सक्षम करें।
