# आरंभ करना

कुछ ही मिनटों में शुरुआत करें।

## 1) एक्सटेंशन इंस्टॉल करें

- [Chrome Web Store](https://chromewebstore.google.com/detail/jabhdcjhdlnppcpbdghnkfkdpfcfleba)
- [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/ai-translator/)
- [मैन्युअल इंस्टालेशन के लिए निर्देश](/hi/installation)

## 2) अपने प्रदाता को कॉन्फ़िगर करें

1. एक्सटेंशन **विकल्प** पृष्ठ खोलें।
2. **बेसिक** (सरल डिफ़ॉल्ट) या **एडवांस्ड** (पूर्ण नियंत्रण) चुनें।
3. अपनी एपीआई कुंजी जोड़ें और सहेजें।

चाबी चाहिए? [प्रदाता और एपीआई कुंजियाँ](/hi/providers) देखें।

## 3) कुछ अनुवाद करें

- टेक्स्ट चुनें → राइट-क्लिक करें → **चयनित टेक्स्ट का अनुवाद करें**
- कहीं भी राइट-क्लिक करें → **अनुवाद पृष्ठ**

यदि कुछ भी गलत लगता है, तो [समस्या निवारण](/hi/troubleshooting) पृष्ठ देखें।
