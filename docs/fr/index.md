## Qu'est-ce que AI Translator ?

AI Translator est une extension Chrome et Firefox qui traduit le texte sélectionné ou
des pages entières à l’aide de fournisseurs d’IA configurables. Vous contrôlez le modèle, le point final,
Clé API et instructions de traduction.
