# Utilisation

## Traduire le texte sélectionné

1. Mettez en surbrillance le texte sur n'importe quelle page Web.
2. Cliquez avec le bouton droit → **Traduire le texte sélectionné**.
3. La traduction apparaît dans une fenêtre contextuelle.

## Traduire une page entière

1. Cliquez avec le bouton droit n'importe où sur la page.
2. Choisissez **Traduire la page**.
3. L'extension remplace le texte de la page par la sortie traduite.

## Afficher le bouton Traduire en ligne

Vous pouvez éventuellement afficher un petit bouton **Traduire** lorsque le texte est sélectionné.
Activez-le dans **Paramètres → Afficher le bouton Traduire lors de la sélection**.
