## Cos'è AI Translator?

AI Translator è un'estensione per Chrome e Firefox che traduce il testo selezionato o
intere pagine utilizzando provider di intelligenza artificiale configurabili. Tu controlli il modello, l'endpoint,
Chiave API e istruzioni di traduzione.
