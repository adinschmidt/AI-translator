# 設定

拡張機能メニューを開き、**Options** をクリックして AI Translator を設定します。

## モード

### Basic モード

- プロバイダー（OpenAI、Anthropic、Google）を選択します。
- API キーを入力します。
- 翻訳先言語を選択します。
- 推奨のエンドポイントとモデルが自動で使われます。

### Advanced モード

- 対応するすべてのプロバイダーから選択できます。
- API エンドポイント、モデル名、翻訳指示をカスタマイズできます。
- セルフホスト環境、OpenAI 互換 API、上級者向け設定に適しています。

## 翻訳指示（Translation Instructions）

**Translation Instructions** を使うと、文体やトーンを指定できます。選択したテキストは自動で追記されます。例:

```
スペイン語に翻訳してください。親しみやすく簡潔なトーンにしてください。
```

## API エンドポイントとモデル

各プロバイダーには、推奨のエンドポイントとモデル名を復元する **Fill Default** があります。必要に応じて、Advanced モードで両方を上書きできます。

## 保存とプライバシー

設定は `chrome.storage.sync` に保存されるため、デバイス間で同期されます。
API キーがコンソールへ出力されることはありません。
