# 使い方

## 選択したテキストを翻訳

1. Web ページ上でテキストを選択します。
2. 右クリック → **Translate Selected Text**。
3. 翻訳結果がポップアップに表示されます。

## ページ全体を翻訳

1. ページ上の任意の場所で右クリックします。
2. **Translate Page** を選択します。
3. 拡張機能がページ内テキストを翻訳結果に置き換えます。

## インラインの Translate ボタンを表示

テキスト選択時に小さな **Translate** ボタンを表示できます。
**Settings → Show Translate button on selection** で有効にしてください。
