# AI Translator プライバシーポリシー

**最終更新日:** 2026年1月10日

## 1. 概要

AI Translator（「当社」「私たち」「本拡張機能」）は、ユーザー自身の AI API
キーを使用してテキストを翻訳できる、クライアントサイドのブラウザ拡張機能です。本拡張機能はブラウザ内で動作し、開発者が所有または運用するサーバーへデータを送信しません。

**本ドキュメントは法的助言ではありません。** 本ポリシーは、本拡張機能によるデータの取り扱いを説明し、利用可否を判断するための情報を提供するものです。

## 2. データの収集と保存

- **個人データ:** 開発者は、個人データ、翻訳テキスト、閲覧履歴を収集・保存・閲覧しません。
- **API キー:** API キーは、ブラウザの同期ストレージ（`browser.storage.sync`）を使用して、お使いのデバイス上にのみ保存されます。ログイン中のブラウザ間で設定は同期されますが、翻訳リクエスト先としてユーザーが選択した AI プロバイダーを除き、開発者や第三者へ送信されることはありません。
- **バックエンドサーバーなし:** 本拡張機能はバックエンドサーバーを運用していません。すべての処理はブラウザ内で行われます。

## 3. 第三者とのデータ共有

本拡張機能は、翻訳機能を提供するために、ユーザーが明示的に選択したテキストを、設定済みの AI プロバイダーへブラウザから直接送信します。本拡張機能を使用することで、以下の各プロバイダーのプライバシーポリシーが適用されることに同意したものとみなされます。

| プロバイダー           | プライバシーポリシー                                                               |
| ---------------------- | ---------------------------------------------------------------------------------- |
| **OpenAI**             | [https://openai.com/privacy](https://openai.com/privacy)                           |
| **Anthropic**          | [https://www.anthropic.com/legal/privacy](https://www.anthropic.com/legal/privacy) |
| **Google (Gemini)**    | [https://ai.google.dev/gemini-api/terms](https://ai.google.dev/gemini-api/terms)   |
| **xAI (Grok)**         | [https://x.ai/legal/privacy-policy](https://x.ai/legal/privacy-policy)             |
| **OpenRouter**         | [https://openrouter.ai/privacy](https://openrouter.ai/privacy)                     |
| **Ollama（ローカル）** | データはローカル端末内に保持され、外部送信されません。                             |

**重要な注意事項:**

- 一部のプロバイダーでは、オプトアウトしない限りデータがモデル学習に利用される場合があります（詳細は各プロバイダーのポリシーを確認してください）。
- OpenRouter は複数の基盤 AI プロバイダーへリクエストをルーティングするため、それぞれ独自のデータポリシーが適用されます。
- 当社は、ユーザーデータを販売・取引・第三者へ移転しません。

## 4. ブラウザストアのポリシー

本拡張機能をブラウザ拡張機能ストアで配布する場合、[Chrome Web Store User Data Policy](https://developer.chrome.com/docs/webstore/program-policies/) や [Firefox Add-on Policies](https://extensionworkshop.com/documentation/publish/add-on-policies/) を含む、適用されるユーザーデータポリシーに準拠します。

## 5. 権限の説明

本拡張機能は以下の権限を要求します。

| 権限                              | 用途                                                                                                     |
| --------------------------------- | -------------------------------------------------------------------------------------------------------- |
| `contextMenus`                    | 右クリックメニューに「Translate」オプションを追加します                                                  |
| `scripting`                       | Web ページ上に翻訳 UI オーバーレイを挿入します                                                           |
| `storage`                         | API キーと設定をローカルに保存します                                                                     |
| `host_permissions` (`<all_urls>`) | 訪問先の任意の Web ページで翻訳結果を表示し、選択した AI プロバイダーへ API 呼び出しを行うために必要です |

## 6. ユーザーによるコントロール

- **削除:** 拡張機能をアンインストールするか、ブラウザの拡張機能データを削除することで、本拡張機能が保存したデータをすべて削除できます。
- **オプトアウト:** ユーザーが翻訳操作を明示的に実行しない限り、AI プロバイダーへデータは送信されません。
- **プロバイダー選択:** 拡張機能設定で選択することで、どの AI プロバイダーにデータを送信するかをユーザー自身で管理できます。

## 7. セキュリティ

- API キーはブラウザの安全なストレージに保存され、AI プロバイダーのエンドポイントへ HTTPS 経由でのみ送信されます。
- 本拡張機能は、翻訳テキストを表示後にログ記録・キャッシュ・保持しません。

## 8. 本ポリシーの変更

本プライバシーポリシーは、必要に応じて更新される場合があります。変更がある場合は、本ドキュメント上部の「最終更新日」に反映されます。
