# Empezando

Ponte en marcha en unos minutos.

## 1) Instalar la extensión

- [Chrome Web Store](https://chromewebstore.google.com/detail/jabhdcjhdlnppcpbdghnkfkdpfcfleba)
- [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/ai-translator/)
- [Instrucciones para la instalación manual](/es/installation)

## 2) Configura tu proveedor

1. Abra la página de extensión **Opciones**.
2. Elija **Básico** (valores predeterminados simples) o **Avanzado** (control total).
3. Agregue su clave API y guárdela.

¿Necesitas una clave? Consulte [Proveedores y claves API](/es/providers).

## 3) Traducir algo

- Seleccione texto → haga clic derecho → **Traducir texto seleccionado**
- Haga clic derecho en cualquier lugar → **Traducir página**

Si algo parece extraño, consulte la página [Solución de problemas](/es/troubleshooting).
