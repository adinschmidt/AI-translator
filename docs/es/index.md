## ¿Qué es AI Translator?

AI Translator es una extensión de Chrome y Firefox que traduce texto seleccionado o
páginas enteras utilizando proveedores de IA configurables. Tú controlas el modelo, el punto final,
Clave API e instrucciones de traducción.
