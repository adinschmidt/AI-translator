# Uso

## Traducir el texto seleccionado

1. Resalte el texto en cualquier página web.
2. Haga clic derecho → **Traducir texto seleccionado**.
3. La traducción aparece en una ventana emergente.

## Traducir una página completa

1. Haga clic derecho en cualquier lugar de la página.
2. Elija **Traducir página**.
3. La extensión reemplaza el texto de la página con el resultado traducido.

## Mostrar el botón Traducir en línea

Opcionalmente, puedes mostrar un pequeño botón **Traducir** cuando se selecciona texto.
Habilite esto en **Configuración → Mostrar botón Traducir en la selección**.
