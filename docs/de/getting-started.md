# Erste Schritte

In wenigen Minuten können Sie loslegen.

## 1) Installieren Sie die Erweiterung

- [Chrome Web Store](https://chromewebstore.google.com/detail/jabhdcjhdlnppcpbdghnkfkdpfcfleba)
- [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/ai-translator/)
- [Anweisungen zur manuellen Installation](/de/installation)

## 2) Konfigurieren Sie Ihren Anbieter

1. Öffnen Sie die Seite **Optionen** der Erweiterung.
2. Wählen Sie **Basic** (einfache Standardeinstellungen) oder **Advanced** (vollständige Kontrolle).
3. Fügen Sie Ihren API-Schlüssel hinzu und speichern Sie ihn.

Benötigen Sie einen Schlüssel? Siehe [Anbieter und API-Schlüssel](/de/providers).

## 3) Übersetze etwas

- Text auswählen → Rechtsklick → **Ausgewählten Text übersetzen**
- Klicken Sie mit der rechten Maustaste auf eine beliebige Stelle → **Seite übersetzen**

Wenn irgendetwas nicht stimmt, schauen Sie auf der Seite [Fehlerbehebung](/de/troubleshooting) nach.
